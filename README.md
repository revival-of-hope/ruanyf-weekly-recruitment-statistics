## Analyze
可以看出来绝大部分工作是不卡硕士学历的,(虽然本科要求211,985也比较卡学历了),而要求硕士学历的基本都是算法岗的,而且都是新兴企业.

总体来说,如果不是铁着头要做算法岗,非要冲所谓高薪的话,反而是本科就业更好,因为本数据集里的很多岗位反而有"三年以上工作经历"这样的经验限制而非学历限制.
