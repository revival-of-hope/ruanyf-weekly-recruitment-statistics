Cryptape 是一家致力于推动区块链技术发展的领先企业，专注于 Nervos CKB（Common Knowledge Base）及其生态系统的核心技术研发。我们坚信去中心化的力量，通过持续的技术创新为全球区块链行业注入活力。自2016年成立以来，我们在区块链领域积累了深厚的技术实力和丰富的实践经验，参与并主导了包括 Ethereum、Nervos CKB、CITA 等多个知名开源项目。我们秉持“Talk is cheap, show me the code”的工程师文化，倡导开放与开源，所有核心项目代码均在 GitHub 上公开，与全球开发者共同构建信任的数字未来。

我们正在为 CKB-VM 团队招聘一位**开发工程师**，这是一个全职远程职位。您将有机会参与 Nervos CKB 虚拟机（CKB-VM）的开发工作，专注于智能合约编译器的开发与优化，推动 CKB 生态系统的性能提升与技术突破。CKB-VM 是一个兼容 RISC-V 指令集的通用虚拟机，支持多种编程语言的脚本开发，是 Nervos CKB Layer 1 的关键组件。详情可查看 https://github.com/nervosnetwork/ckb-vm.

- 参与开发智能合约编译器，提升 CKB-VM 的性能、兼容性和安全性。
- 优化现有编译器工具链，解决 C/C++ 到 CKB-VM 指令集的转换难题。
- 使用 Rust 语言实现高性能、低代码的编译器模块。
- 参与团队技术讨论与代码审查，贡献技术文档及知识分享。
- 与生态开发者合作，确保编译器满足多样化的智能合约开发需求。

- **必备条件**：
    - 计算机科学、软件工程或相关领域的扎实基础（本科及以上学历）。

    - 愿意学习并使用 Rust 进行开发（有 Rust 经验者优先）。
    - 对区块链技术及虚拟机逻辑（CKB-VM 或类似）有浓厚兴趣并愿意深入研究。
    - 良好的逻辑思维和问题解决能力，能够独立承担项目任务。
- **加分项**：
    - 有编译器开发、虚拟机优化或低级系统编程经验。
    - 熟悉 RISC-V 指令集或相关汇编语言。
    - 参与过开源项目，或有区块链相关开发经验。
    - 了解密码学、智能合约或 Nervos CKB 生态系统。
