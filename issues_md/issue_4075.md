# 字节跳动 Web Infra 团队招聘啦

## 🏄 我们是谁

我们是字节跳动 Web Infra - Web Solutions 团队，服务于整个公司的 Web 生态，我们的愿景是 **打造世界一流的 Web 生态技术体系，为字节产品提供极致的用户体验和开发体验**。

我们坚信 Web 技术是最伟大的事情之一，Web Infra 团队竭力提供更好用的工具，让 Web 开发更加容易、让开发者能收获更好的开发体验、用户收获更好的用户体验。与此同时，开源也一直是我们在长期探索的事情。

我们打造了一系列 Web 工具来提升开发效率和体验，包括但不限于：

- **基于 Rust 的 Web 构建工具** —— [Rspack](https://github.com/web-infra-dev/rspack)，旨在建设高性能前端工具链，让跨端和 Web 场景拥有 “快人一步” 的开发体验，具备极速的编译和热更新性能。
- **基于 Rspack 的开源方案**，包括 [Rsbuild](https://github.com/web-infra-dev/rsbuild)、[Rspress](https://github.com/web-infra-dev/rspress) 和 [Modern.js](https://github.com/web-infra-dev/modern.js)，构成一系列开箱即用的解决方案，提供从 Web 构建、静态站点生成到全栈研发的多场景支持。
- **高性能 Web 解决方案**，基于 Web 不止于 Web。突破传统的 WebView，与端、浏览器内核相结合，持续探索各种端性能优化方式，让开发者以极低成本享受优化能力。
- **现代 Web 工程体系**，包括基于 React 的渐进式 Web 开发框架、基于 esbuild 的模块开发工具、monorepo 解决方案、微前端/微模块解决方案、构建诊断分析工具。

当前，这些工具在 ByteDance 内部被广泛使用和好评。同时，其中多个项目已经开源到 GitHub，与社区开发者共同建设和发展。

> # 👋 [Flexport](https://cn.flexport.com/company/careers/) （外企，新增岗位）
> ## 关于Flexport
> Flexport，科技货运领域的独角兽公司，成立于2013年。Flexport 通过其研发的国际货运平台系统，为客户提供海陆空货运的透明化服务和经过授权的代理经纪人服务，被称为国际货运界的“Uber”。客户可以通过Flexport平台预定货船、卡车、货运飞机来满足其对国际供应链物流服务的需求。目前Flexport已经在为全球超过10,000家客户和供应商提供运营和管理供应链物流的服务，比如Apple，Logitech，Lululemon，Peloton等。2019年2月，Flexport完成新一轮由软银愿景基金领头的10亿美元的融资，当时估值32亿美元。你可以通过 An App For Global Trade对Flexport的创立初衷有所了解。
> 
> Flexport believes that nothing helps people improve their lives more than the ability to trade with one another. And, yet international shipping is still way too cumbersome. So Flexport built an app to make it easier. If Flexport succeeds, nearly everything you buy will cost less.
> 
> ## 为什么选择Flexport
> 公司发展空间： 2019年公司估值32亿美元，目前的实际估值已经增长了很多。作为货运领域的独角兽科技公司，随着一系列公司长远战略的实施，未来的增值空间也是巨大的。 竞争力的薪酬：作为人才吸引的重要组成，Flexport提供对标国内大厂的薪酬待遇，薪资结构包含Base + Bonus + 股票期权。并且授予股票的价格要比目前市场的价格低很多潜在的股票价值增长是很客观的。 人才组成：随着公司的发展，越来越多的来自其他大厂（Amazon，Google，Facebook等）的人才加入到Flexport，以及很多有海外背景的同学回国加入中国的研发团队，一起为Flexport的壮大发光发热，实现公司和个人的共同发展 优越的福利：公司为员工提供免费的高端医疗保险；提供免费午餐和各种零食；Flexbile PTO等等。 Culture：作为硅谷的独角兽公司，即便在中国，也保留着硅谷的工作方式和文化，如果你是从海外回国的同学，你会很顺利的融入到开放、多元、自由的工作氛围中。 工作内容：作为科技货运的先驱者，所面临的技术挑战是前所未有的。不论是从产品战略和技术架构等方面，都需要通过创造性的解决问题来达到行业变革的目的。其中，作为最接近国际供应链和物流重地（中国）的团队，Flexport中国技术团队会对Flexport国际货运平台的建设起到至关重要的作用。 工作时间：这里没有996，没有内卷。弹性的工作时间，让你的Work-Life-Balance能得到充分保障。
